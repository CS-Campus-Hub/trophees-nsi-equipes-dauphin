### Pré-requis

Ce qu'il est requis pour commencer avec votre projet...

- Python
- Pip
- PyGame

### Installation

Ouvrez un terminal et vérifiez d’abord que Python et Pip sont bien installés en écrivant :

```bash
python --version
pip --version

## Démarrage

Pour lancer le projet, ouvrez le fichier main.py puis exécutez le code. Une fois la fenêtre ouverte, appuyez sur “Play” pour commencer le quiz. Utilisez ensuite les touches → ↑ ← ↓ pour vous déplacer et accéder aux différentes questions.

## Fabriqué avec

- Python avec le module PyGame pour la programmation.
- Krita et IbisPaint pour l'art.
